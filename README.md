# Openpot
The privacy-first, zero-knowledge session tracker.

# Core

### 🛡️ The Zero-Knowledge Mission
Openpot is a sovereign session tracker built for the global cannabis community. 
Our architecture is designed to be **legally invisible** and **mathematically anonymous**.

**Core Engineering Principles:**
1. **No Server-Side Identity:** We do not store names, emails, or IDs.
2. **Local-Only State:** User history lives only on the user's device.
3. **Aggregated Metrics:** We use anonymized hashtable counters to measure success, not tracking.
4. **Stuttgart-Safe:** Designed to comply with 2026 German privacy standards by ensuring no PII ever touches our infrastructure.
